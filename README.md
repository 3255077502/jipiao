# 机票购买管理系统
项目名称：机票购买管理系统。类型：Web网站。技术栈：vue，vue-cil，vue全家桶，JavaScript；spring boot，mybatis，spring MVC，MySQL;用户可以购买机票，购买餐饮服务，视图化选座；管理员可在后台管理系统查看用户信息等。
